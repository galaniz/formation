# setDisplay  

**<code>setDisplay(item: Element | null, type: &#39;show&#39; | &#39;hide&#39; | &#39;focus&#39;, attr?: string | undefined): number</code>**  

Toggle display and focus of element.

## Parameters  
- **`item`** <code>Element | null</code> required  
- **`type`** <code>&#39;show&#39; | &#39;hide&#39; | &#39;focus&#39;</code> required  
- **`attr`** <code>string | undefined</code> optional

## Returns  

<code>number</code>
